🚩 Signalements

Signalements envoyés par les utilisateurs depuis les profils, commentaires, messages chat et photos. Utilise les filtres ci-dessous pour les trier.

💰 Portefeuilles providers

Solde du portefeuille des providers (pet-sitters + walkers). Le provider est crédité de 80% du tarif au début de la mission ; il peut ensuite utiliser ce solde pour la boutique ou le retirer vers son IBAN. Les retraits non éligibles (IBAN non vérifié) restent en attente.

💬 Modération du chat

Conversations récentes avec l'historique complet. Les admins peuvent supprimer un message abusif (soft-delete) ; le message reste visible ici pour l'audit, mais s'affiche comme « Message supprimé » dans l'app.

🏆 Avantages fidélité

Owners Premium (10+ réservations) · Top Sitters (20+ prestations + note 4.5★) · Top Walkers (20+ balades + note 4.5★). Commission réduite à 15% pour Top Sitter/Walker.

💰 Mes revenus & retraits

Tes commissions HoPetSit (20% sur réservations) + paiements boutique (Boost, PawSpot, Premium, Chat add-on).
La card verte « Solde retirable » ci-dessous est le montant que tu peux envoyer vers ton IBAN société d'un clic.

🐛 Bug Reports

Tous les bugs signalés par les utilisateurs depuis leur profil. Un email part automatiquement à hopetsit@gmail.com à chaque signalement.

🚀 Activité boutique

Tous les achats boutique (Boost profil, Map Boost, Premium, Chat add-on) pour pet-sitters, owners et walkers. Revenus, achats actifs, historique.

📣 Annonces & modération

Toutes les annonces (demandes de réservation et posts média). Ban masque l'annonce côté utilisateurs tout en gardant les données en DB (audit + analytics). Supprimer est définitif.

🗺️ PawMap analytics

Vue d'ensemble des POIs (vétos, parcs, points d'eau...) et des signalements communautaires (chien perdu, danger, point d'eau, etc.). Tous les signalements sont conservés — même après expiration 48 h — comme dataset historique. Utile pour repérer les hotspots et détecter les abus.

🌱 Alimenter la carte (OpenStreetMap)

Charge automatiquement les POIs depuis OpenStreetMap pour un pays ou toute l'Europe. Les catégories couvrent vétos, parcs à chiens, boutiques, toiletteurs, points d'eau, hôtels/restos pet-friendly. Le seed tourne en tâche de fond (1-3 min par pays). Clique "Tout l'Europe + Suisse" pour charger les 29 pays d'un coup.
